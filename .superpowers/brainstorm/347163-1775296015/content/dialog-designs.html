Component notes

  • NewWorktreeDialog — opens from "+ new worktree" per repo. Repo is pre-selected. Base ref select fetches branches via get-branches IPC. Install toggle reads from .wt config.
  • CleanWorktreesDialog — opens from "clean" button in action bar. Cross-repo view. Groups by repo. Checkbox multi-select. Reason badges (merged / remote deleted).
  • ConfirmDialog — reusable for remove worktree, kill session. Shows context + optional "also delete branch" checkbox.
  • All dialogs use shadcn Dialog, Select, Checkbox, Button components backed by Base UI primitives.
  • Theme — Tailwind CSS custom properties mapped from Omarchy colors. shadcn components inherit the theme automatically.
